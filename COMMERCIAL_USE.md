# Commercial Use

OpenMemo is open-source software released under the AGPLv3 license.

This means:

- You can freely use, modify, and redistribute the software.
- If you deploy OpenMemo as a network service with modifications, those modifications must also be released under AGPLv3.

Commercial services based on OpenMemo must comply with AGPLv3 requirements.

## Trademark

The name "OpenMemo" and related branding may not be used for commercial products or services in a way that suggests official affiliation with the original project.

If you plan to build commercial offerings using OpenMemo, please ensure that your branding clearly distinguishes your project.
